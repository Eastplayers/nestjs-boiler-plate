## Description

A NestJS project of Eastplayers

## Installation

```bash
$ yarn install
```

## Running the app

```bash
# development
$ yarn run start

# watch mode
$ yarn run start:dev

# production mode
$ yarn run start:prod
```

## Running the crud generator script
```bash
$ bash crud.sh <module_name>
```
Example:
```bash
$ bash crud.sh cateogry

$ bash crud.sh product

$ bash crud.sh "product category"
```
